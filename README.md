 # Druhan's dotfiles

 These are personal configuration files for my Arch Linux system using QTile on X11.  
 They do not include configurations for the WM and for Neovim. Those are instead located in repositories of their own for better management. See [Qtile](https://github.com/DruhanShah/qtile-config) and [Neovim](https://github.com/DruhanShah/nvim-config).
